## ● Tell me how to remove them locally and remotly.

locally
```
    git branch -d test
    git branch -d dev
```

remote
```
    git push origin --delete test
    git push origin --delete dev  
```

## ● tell me how to delete tag locally and remotely.
locally
```
    git tag -d v1.0
```
remote
```
    git push origin --delete v1.0
```
